# Roadmap para Programador Java Júnior
```
├── 1. Aprenda Spring Boot
│   ├── Spring Data
│   │   ├── Como criar um Repository
│   │   ├── Como criar um Service
│   │   └── Queries personalizadas com @Query
│   ├── Spring Web
│   │   └── Como criar um Controller
│   ├── Estereótipos no Spring
│   │   ├── @Component / @Service / @Repository
│   │   └── @RestController vs @Controller
│   ├── Mapeamento de Entidades
│   │   ├── @Entity, @Id, @GeneratedValue
│   │   ├── Mapeamento de relacionamento
│   │   │   ├── @OneToOne
│   │   │   ├── @OneToMany / @ManyToOne
│   │   │   └── @ManyToMany
│   ├── Injeção de Dependências
│   │   ├── Como injetar dependências
│   │   ├── @Autowired vs construtor
│   │   └── Dependência cíclica
│   └── Criação de Testes Unitários
│       ├── Mockito
│       └── Spring Test

├── 2. Aprenda Banco de Dados
│   ├── SQL
│   │   ├── DML
│   │   │   ├── INSERT
│   │   │   ├── UPDATE
│   │   │   └── DELETE
│   │   └── DDL
│   │       ├── Criar tabela
│   │       ├── Definir Primary Key
│   │       └── Constraints (FK, UNIQUE, NOT NULL)
│   └── PostgreSQL ou MySQL

├── 3. Aprenda Angular (Frontend opcional)
│   ├── Modules
│   │   ├── Estrutura
│   │   └── Importações
│   ├── Componentes
│   │   ├── Criação e template
│   │   └── Data Binding
│   ├── Serviços
│   │   └── Comunicação com API REST (HttpClient)
│   └── Rotas e navegação

├── 4. Git e Versionamento
│   ├── Git init / add / commit / push / pull
│   ├── GitHub: criação de repositórios
│   └── Branches e Pull Requests

├── 5. Boas Práticas com Java
│   ├── Clean Code
│   ├── SOLID
│   └── Padrões de Projeto básicos
│       ├── Singleton
│       └── Factory

├── 6. Documentação e Teste de API
│   ├── Swagger
│   ├── Postman
│   └── Testes de integração

└── 7. Projetos Práticos (Portfólio)
    ├── API de Produtos com CRUD
    ├── Sistema de autenticação simples
    └── Integração com frontend (Angular ou React)
